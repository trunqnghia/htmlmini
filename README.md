# Mini-project 2: HTML Head Checklist

Dự án trang web hướng dẫn cấu hình thẻ `<head>` chuẩn chỉnh trong HTML5, tuân thủ các quy tắc tổ chức dự án chuyên nghiệp, viết code sạch (clean code) và phân vùng UI rõ ràng.

## 🛠 Tính năng và Kiến thức áp dụng
- Cấu hình chuẩn các thẻ: `charset`, `viewport`, `title`.
- Cách liên kết file CSS và nhúng JavaScript chuẩn hiệu năng (`defer`).
- Quy chuẩn thụt dòng (Indentation) và ghi chú (Comment) phân vùng UI.
- Xây dựng bố cục Layout Responsive (Flexbox).

## 📐 Sơ đồ phân vùng UI & Thiết kế
> Ảnh thiết kế chi tiết được lưu trữ tại thư mục `/assets/ui-design.png`.

Giao diện HTML được phân chia thành 4 phân vùng chính dựa trên bản vẽ thiết kế:
1. **Header:** Thanh tiêu đề của trang checklist.
2. **Main Content:** Khu vực hiển thị nội dung chi tiết các thẻ cấu hình head.
3. **Sidebar:** Nơi đính kèm ảnh chụp thiết kế UI và chú thích cấu trúc file.
4. **Footer:** Thông tin bản quyền tác giả.
